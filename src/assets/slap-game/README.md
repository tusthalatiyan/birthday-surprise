# Cheer-up game photo

Drop one image file into this folder and the cheer-up game will use it automatically.

Supported formats:
- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.avif`

If you add multiple files, the app prefers names starting with `photo.`, `slap-game-photo.`, or `cheer-up-photo.`. Otherwise it uses the first file alphabetically.
